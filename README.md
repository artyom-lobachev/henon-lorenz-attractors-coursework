# Код к курсовой работе

Репозиторий содержит исходный код, использованный при выполнении курсовой работы на тему:

**«Псевдогиперболический спиральный аттрактор в моделях с дискретным временем»**

## Описание

В работе численно исследуется трёхмерное отображение Эно:

x_{n+1} = y_n  
y_{n+1} = z_n  
z_{n+1} = M_1 + Bx_n + M_2y_n - z_n^2

Код использовался для построения фазовых портретов, вычисления неподвижных точек, расчёта мультипликаторов, показателей Ляпунова и построения карт параметров.

## Содержимое репозитория

Основные файлы находятся в папке `src`.

- `henon3d_analysis.py` — расчёт неподвижных точек, матрицы Якоби, мультипликаторов и показателей Ляпунова для отдельных наборов параметров.
- `lyapunov_maps.py` — построение карт показателей Ляпунова и карты динамических режимов на плоскости параметров.
- `lambda1_with_bifurcation_curves.py` — наложение бифуркационных кривых на карту старшего показателя Ляпунова.
- `bifurcation_curves_overview.py` — построение обзорной бифуркационной диаграммы на плоскости параметров.

## Используемые библиотеки

Для запуска программ нужны библиотеки NumPy и Matplotlib.

Установить их можно командой:

`pip install -r requirements.txt`

## Порядок запуска

Сначала можно выполнить расчёт для отдельных наборов параметров:

`python src/henon3d_analysis.py`

Затем можно построить карты показателей Ляпунова:

`python src/lyapunov_maps.py`

После этого можно наложить бифуркационные кривые на карты:

`python src/lambda1_with_bifurcation_curves.py`

Обзорная бифуркационная диаграмма строится отдельно:

`python src/bifurcation_curves_overview.py`

## Примечание

Репозиторий является приложением к курсовой работе и содержит программную реализацию численных экспериментов. Полученные изображения и файлы с данными могут отличаться в зависимости от выбранного размера сетки, числа итераций и параметров расчёта.
